🚀 INSTALAÇÃO SUPER RÁPIDA:

WINDOWS:
1. Baixe e instale (com internet):
   • Python: https://python.org (MARCAR "Add to PATH")
   • Node.js: https://nodejs.org  
   • MongoDB: https://mongodb.com/download-center/community

2. Execute: INSTALAR.bat
3. Execute: INICIAR.bat  
4. Acesse: http://localhost:3000

LINUX:
1. sudo apt install python3 nodejs mongodb
2. ./install.sh  
3. ./start.sh
4. Acesse: http://localhost:3000

🌐 USO EM REDE LOCAL:
• Execute: INICIAR-REDE.bat
• Acesse de outros PCs: http://IP-DO-SERVIDOR:3000
